# 🏎️ Nitro Racer

[![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/ISC)
[![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=flat&logo=vite&logoColor=white)](https://vitejs.dev/)

A high-octane, browser-based 3D racing game. Drift, boost, and beat your AI rivals without ever leaving your browser.

## ✨ Features

- **3D Graphics**: Immersive track environments with diverse skyboxes (Night, Rain, Snow, Summer).
- **AI Rivals**: Compete against 3 competitive AI racers.
- **Nitro Boost**: Use your boost strategically to secure first place.
- **Dynamic Camera**: Switch between Chase Cam (3rd Person) and Cockpit (1st Person).
- **Customizable Experience**: Adjustable camera shake intensity and immersive post-race cinematics.
- **Zero Installation**: Play directly in your web browser!

## 🚀 Getting Started

### Prerequisites

You will need [Node.js](https://nodejs.org/) installed on your machine to run the local development server.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DZ1shetty/NitroCar.git
   cd NitroCar
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the provided localhost URL (typically `http://localhost:5173`).

## 🎮 Controls

- **W / Up Arrow**: Accelerate
- **S / Down Arrow**: Brake / Reverse
- **A / Left Arrow**: Steer Left
- **D / Right Arrow**: Steer Right
- **Space**: Handbrake / Drift
- **Shift / N**: Nitro Boost
